com.choosemuse.libmuse.CharacteristicId.MUSE_SERVICE

The id corresponding to the UUID for the Muse service.

This is technically not a characteristic, but we keep it here so we can get its UUID in CharacteristicMapper.

This is the first enum value after the last valid characteristic; i.e. the code should be able to enumerate all valid characteristics by iterating over the range [0, muse_service).

Left auxiliary electrode measurement.

This can be either left or left_bypass depending on the configuration. The difference is that left_bypass has no gain applied to it.
