Detailed Description

Advertised information about a device.

These values are available prior to connection, and can be used to decide which devices are available to connect to.

String com.choosemuse.libmuse.DeviceInformation.getIdentifier ( )
inline

This value is platform-specific. On iOS, it is the UUID provided by CoreBluetooth. On Android, it is the MAC address.

double com.choosemuse.libmuse.DeviceInformation.getLastDiscoveredTime ( )
inline

Time at which this device was most recently discovered.

This value is in microseconds since some common start time (e.g. unix epoch, device start time, or app start time). It should not be used to record absolute times, but only compared with times from other devices to determine which was discovered most recently.
